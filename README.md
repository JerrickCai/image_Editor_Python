# image_Editor_Python

### implementation of a simple photo editor

_The goal of this implementation is to practice the Gui of python using **Tkinter** and the image processing using **opencv2** and **pillow**._



Some functionalities:

- Reset
- Zoom in
- Zoom out
- Flip horizontally
- Flip vertically
- Rotate
- Brightness
- Export
